# 📊 Tracker de Hábitos Personalizado

Um aplicativo web otimizado para celular que permite rastrear e gerenciar suas atividades diárias em três categorias principais: Clientes, Categorias e Atividades/Rotina.

## 🎯 Objetivo

Manter a entropia baixa nos pontos mais importantes da vida através do acompanhamento diário de tarefas e visualização de desempenho.

## ✨ Funcionalidades

### 📅 Rastreamento Diário
- Anote demandas diárias para cada item
- 9 status diferentes com emojis visuais
- Navegação fácil entre dias

### 📜 Histórico
- Visualize o histórico das últimas semanas/meses
- Veja o que foi feito em cada dia
- Compare o desempenho ao longo do tempo

### 📈 Relatórios de Desempenho
- Visualização semanal, mensal, anual e geral
- Taxa de conclusão por categoria
- Gráficos de progresso
- Estatísticas detalhadas

## 🎨 Status Disponíveis

- 🟡 Em andamento
- 🟣 Bloqueado
- 🟢 Concluído
- 🔵 Aguardando
- 🔴 Não feito
- ⚪ Pular

## 🚀 Como Usar

1. Abra o arquivo `index.html` em seu navegador
2. Para cada dia, clique nos itens e selecione o status apropriado
3. Use os botões de navegação para ver histórico e relatórios
4. Os dados são salvos automaticamente no navegador

### 🎤 Notas por voz

- Cada linha/item tem um botão de microfone (🎙️). Toque no botão para iniciar a gravação de voz.
- Dê permissão ao navegador para acessar o microfone quando solicitado.
- Ao terminar, toque novamente para parar; a transcrição será automaticamente adicionada à nota do item com carimbo de hora.
- Observação: a transcrição usa a Web Speech API (SpeechRecognition). Nem todos os navegadores suportam — recomendamos Chrome ou Safari atualizados.

## 💾 Armazenamento

Os dados são armazenados localmente no navegador usando LocalStorage. Nenhuma informação é enviada para servidores externos.
